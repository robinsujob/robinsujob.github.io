2.3 生成艺术字

字体是一种艺术形式，不仅可以传递信息，还可以为品牌和产品带来独特的风格和个性。如何选择合适的字体成为了品牌宣传和市场营销中的重要环节。

Stable Diffusion可以将文字和图像完美地融合在一起，形成了一种全新的表现方式。 这种创意让字与图相互交织，呈现出独特的视觉效果！本文主要分享使用伊克罗德的 imAgine 产品生成光影艺术字的最佳实践。我们将以 QRCode 作为 ControlNet 的输入，使 QRCode 文字图像融入到艺术图像中，从而以一种全新的方式来表达和传递信息。以下为几张图片案例：

  • 渲染海报艺术字
  • 嵌入艺术字
  • 光影效果艺术字

前期准备： (1)底图

首先需要准备类似我下面的白字的图片，白字的内容决定了画面里展示的内容。推荐使用PS、美图秀秀，亦可以打开Word打字然后截图也行。

前期准备：(2)模型环境准备

当我们选择好合适的底图文件进行后续加工后，选择合适的模型生成至关重要。如果目标是生成有质感的服装搭配图片,我强烈建议使用3D模型,例如 MajicmixRealistic和 ChilloutMix。这些模型可以增强图像的逼真性和质感。 具体来说,我推荐根据图片内容选择如下模型:

  • 真人图片: majicmixRealistic、chilloutmix ，这些模型在处理人像图像方面表现优秀。
  • 其他类型图片: revAnimated、realisticVision、deliberate ，这些通用模型可以处理各类图像,输出效果更为均衡。

1.模型选择与参数调整

由于是关于真实物品的图片，所以在推荐模型里选择revAnimated作为主模型 我们的需求是，生成”文字”以白色浪花的形式嵌入海洋的图片,故在这里推荐模型里选择了revAnimated作为主模型. 填好提示词后就需要调整参数，主要分为Stable Diffusion参数和ControlNet参数

Part1: Stable Diffusion参数
  • 采样方法(Sampler)：Euler
  • 宽度(Width)和高度(Height)则根据底图的大小调整 - 由于图片大小我选择1270（长）*520（高）
  • 其他参数则保持默认
Part2: ControlNet参数
  • 海报效果需要白底黑字的底图
  • 把底图上传到ControlNet内，如下图所示
  • 勾选启用(Enable)和完美像素模式(Pixel Perfect)
  • 预处理器(Preprocessor)选择depth_midas
  • **模型(Model)**选择depth
  • **控制权重(Control Weight)**建议0.7-1之间 ，同时可以适当调整引导介入时机0-0.1

1.模型选择与参数调整

由于是关于真实场景的图片，这里依旧推荐模型里选择revAnimated作为主模型 我们的需求是，生成”文字”以白色浪花的形式嵌入海洋的图片,故在这里推荐模型里选择了revAnimated作为主模型. 填好提示词后就需要调整参数，主要分为Stable Diffusion参数和ControlNet参数

Part1: Stable Diffusion参数
  • 采样方法(Sampler)：DPM++3M SDE Karras
  • 选代步数(Steps)：40
  • 勾选完美像素模式(Pixel Perfect)
  • 宽度(Width)和高度(Height)则根据底图的大小调整,其他参数则保持默认.
Part2: ControlNet参数
  • 勾选启用(Enable)和完美像素模式(Pixel Perfect)
  • 预处理器(Preprocessor) 选择Tile_resample
  • 模型(Model) 选择Tile
  • 控制权重(Control Weight) 建议0.65-0.9之间,其他参数则保持默认.

1.模型选择与参数调整

由于是关于真实场景的图片。这里对于推荐模型里选择revAnimated作为主模型 我们的需求是，生成天津"的城市景观中嵌入字体的图片,故在这里推荐模型里选择了revAnimated作为主模型. 填好提示词后就需要调整参数，主要分为Stable Diffusion参数和ControlNet参数

Part1: Stable Diffusion参数
  • 采样方法(Sampler)：DPM++3M SDE Karras
  • 选代步数(Steps)：40
  • 勾选完美像素模式(Pixel Perfect)
  • 宽度(Width)和高度(Height)则根据底图的大小调整,其他参数则保持默认.
Part2: ControlNet参数
  • 勾选启用(Enable)和完美像素模式(Pixel Perfect)
  • 预处理器(Preprocessor) 选择None
  • 模型(Model)选择QRCode_Monster

截至目前，QRCode Monster 是我们测试后认为成功率最高，也是文字融入图像效果最好的模型，目前 QRCode Monster 已经更新至 v2 版本，相比起 v1，融合度和出图率会有更高的表现。

  • 控制权重(Control Weight) 建议1.15-1.5之间,其他参数则保持默认.
